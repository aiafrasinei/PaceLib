How to install

***

This is the recommended way to use PaceLib.

Prerequisites:
- c++20 compiler
- SDL3, SDL3_image, SDL3_ttf 
- cmake

Until SDL3 is released:

https://github.com/libsdl-org

Build from source with cmake and make install.

**Linux**:

make install to /usr/local

**Windows**:

Install msys2

https://www.msys2.org/


**MacOS**:

Install brew
